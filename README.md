# Crypto example
This example to show:
* How to Hash
* Encrypt plain text and Decrypt cipher text
* Signature and Verify plain text.


## Require modules:

* sha2 = "0.10.0"
* rsa = "0.5.0"
* rand = "0.8.4"
* hex-literal = "0.3.4"

## How to run
* cargo build
* cargo run

## How to run test
* cargo test
